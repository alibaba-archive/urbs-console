# urbs-console
> Urbs 灰度平台管控后台应用

## Features
+ 方便的灰度管理控制界面
+ 支持按用户、群组、比例灰度
+ 支持 ACL，管理后台用户权限
+ 支持操作日志
+ 支持三方实现用户身份验证，便于集成已有系统

## Depends
- urbs-setting >= 1.3.0
